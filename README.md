# All-in-One QR Tool

A **single-page web application** that combines a **QR Code Generator**, **QR Code Designer**, and **Link Shortener**. Built with **HTML, CSS, JavaScript**, and **Bootstrap**, this tool allows you to create, customize, and download QR codes directly in the browser, without any backend or Node.js requirements.

---

## Features

### 1. QR Code Generator
- Generate **single or multiple QR codes** from URLs.
- Download generated QR codes as PNG.
- Supports multiple QR codes at once with easy UI.

### 2. QR Code Designer
- Customize QR codes with **size, foreground color, and background color**.
- Upload an existing QR PNG to edit or overlay an icon.
- Add **preloaded icons** (Instagram, YouTube, Snapchat, TikTok) or **upload your own**.
- Live preview updates instantly when changing colors, size, URL, or icon.
- Download the customized QR code directly as PNG.

### 3. Link Shortener
- Generate simple, random shortened links in-browser.
- Copy shortened links to clipboard with a single click.
